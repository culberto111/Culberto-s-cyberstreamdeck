# Culberto's CyberStreamDeck

A mobile production and streaming system built around an Android tablet, Linux, and multiple Android phones.

## Main Goals

- One-tap Linux startup
- One-tap Linux shutdown
- Cyberdeck-style loading screen
- DeckOS control dashboard
- Phone #2 switchboard and Wi-Fi camera
- Local stream preview on Phone #1
- Twitch streaming tools
- DJ, MPC, DAW, and looping workspace
- Backup and recovery system

## Device Roles

### Tablet
- Main workstation
- Android and Debian Linux
- DJ, MPC, DAW, and streaming tools
- Screen capture and video mixing
- DeckOS interface

### Phone #1
- Local stream preview
- Twitch chat and stream monitoring

### Phone #2
- Tablet switchboard
- Wi-Fi camera
